# SSM-Maven
本项目使用IDEA，利用Maven管理项目，整合SSM框架搭建论文管理系统，实现简单的增、删、改、查功能<br/>
使用步骤：<br/>
1.下载项目，导入idea编辑器。<br/>
2.下载sql数据库文件，导入到你的数据库中。<br/>
3.运行，不要忘了将jdbc.properties中的数据库密码和数据库名改为你自己的。<br/>
注意：在配置tomcat服务器时图中位置不要有项目名（或自己更改项目跳转路径），否则url跳转不正确，报错404。<br/>
